# 분할정복기법



분할 : 해결할 문제를 여러개의 작은 부분으로 나눔

정복 : 나눈 작은 문제를 각각 해결함

통합 : (필요하다면) 해결된 해답을 모음



<퀵정렬과 병합정렬의 다른점>

퀵정렬은 기준 아이템(피봇) 중심으로 분할, 병합정렬은 그냥 단순히 두부분으로 나눈다

퀵정렬은 후처리 작업 필요없고 병합정렬은 '병합'이라는 후처리 작업이 필요



피봇 - 왼쪽 끝, 오른쪽 끝, 임이의 세개 값 중에 중간값

교차되면 j랑 피봇이랑 변경
